Welcome to the Student Management System

This system helps in managing student data efficiently. Below are the main features:


  • View All Students: Click this button to see a list of all students currently in the database. You can view details like their name, roll number, and other relevant information.
  • Add Student: Use this feature to add a new student to the system by providing their details.
  • Remove Student: This option allows you to remove a student from the database by specifying their roll number or selecting them from the list.
  • Search Student: This feature helps you search for a specific student based on criteria such as first name and last name, or any other information available in the system.

Please explore the options above to manage students effectively!
